## 9. Compte de démo à créer AVANT la soumission (action Johan)

1. Sur www.allojoe.fr : créer un compte `demo@allojoe.fr` avec un mot de passe dédié (pas un mot de passe utilisé ailleurs)
2. Lui attribuer un numéro actif + payer l'abonnement (ou l'activer via le back-office admin : compte → statut « active » + attribuer un numéro)
3. Laisser 2-3 messages vocaux de démo sur le numéro (pour que le répondeur/transcriptions aient du contenu)
4. Reporter le mot de passe dans le champ « App Review Information »

## 10. Captures d'écran (à faire sur ton iPhone, format 6,7" — iPhone 15 Pro Max ou équivalent)

Apple demande 3 à 10 captures. Ordre conseillé (le 1er visuel vend) :
1. **Accueil** — ligne active, nom de l'entreprise, appels récents
2. **Répondeur** — un vocal avec badge 🚨/🛠️, résumé IA et transcription visible
3. **Appel entrant** — l'écran d'appel avec le nom du client (fais-toi appeler pour la capture)
4. **Réceptionniste IA** — l'écran de réglages avec messages d'accueil
5. **Statistiques** — les nouvelles stats avec filtres de période
6. **Inscription** — l'écran « Choisissez votre numéro »

Astuce : mets le téléphone en Ne pas déranger, batterie pleine, 9h41 si tu veux faire puriste 😄. Captures = Volume + Power, puis AirDrop sur le Mac.
